# Web-Technologies-VibeFit-Project
VibeFit – a responsive multi-page fitness website built with HTML, CSS Grid, and Flexbox for the Web Technologies course at UDST. The project showcases a clean, user-friendly design with structured navigation, featured products, testimonials, and a subscription section — reflecting core front-end development and web design principles.
